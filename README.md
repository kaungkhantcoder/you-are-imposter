This app is finding who is imposter in the mactch. I created for family, inspired from other
imposter game. However, in my game, that can use with Burmese Language. Fun Game.

Follow these steps:

```sh
# Step 1: Clone the repository using the project's Git URL.
git clone <https://github.com/kaungkhantcoder/you-are-imposter.git>

# Step 2: Navigate to the project directory.
cd <you-are-imposter>

# Step 3: Install the necessary dependencies.
npm i / npm install

# Step 4: Start the development server with auto-reloading and an instant preview.
npm run dev
```

## What technologies are used for this project?

This project is built with:

- Vite
- TypeScript
- React
- shadcn-ui
- Tailwind CSS
